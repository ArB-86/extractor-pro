# Extractor-Pro

Extractor-Pro is a production-grade educational content extraction engine and dataset pipeline for NCERT Mathematics books.

The repository currently contains:

- the extraction pipeline and parser implementation,
- dataset generation and packaging scripts,
- evaluation utilities,
- and local website scaffolding work in progress.

## Current status

The working tree includes active HPC-oriented extraction jobs and generated dataset artifacts that are intentionally not committed here.

## Suggested local setup

1. Create a Python virtual environment.
2. Install dependencies from the project requirements.
3. Run the extraction and packaging scripts as needed.

## Notes

This repository is being used to build both the extraction engine and supporting tooling for the future ShikshaVault website.
